# Learning Three.js: 3D Interactive Case

A basic Three.js scene with HDRI lighting, orbital controls, and a GLB 3D model.

## Folder Structure

- `public/`: Static assets like 3D models (`.glb`/`.gltf`) and textures.
- `src/`: Application source code, including `main.js` and `style.css`.

## Learning Three.js: 3D Interactive Case

An interactive 3D web application built while learning **Three.js** and **Vite**. This project renders a highly detailed, textured 3D wooden kit/case asset in a real-time web environment.

## Features
* **Real-time 3D Rendering:** Utilizing Three.js WebGL renderers.
* **Asset Loading:** Asynchronously loads complex `.gltf` / `.glb` 3D models with textures.
* **Orbit Controls:** Allows the user to rotate, pan, and zoom around the object seamlessly.
* **Responsive Canvas:** Automatically updates resolution and aspect ratio on window resize.

## Tech Stack
* **Core:** JavaScript, HTML5, CSS3
* **3D Library:** [Three.js](https://threejs.org/)
* **Build Tool/Bundler:** [Vite](https://vitejs.dev/)
